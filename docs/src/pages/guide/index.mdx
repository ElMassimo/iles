## Usage ðŸš€

The `iles` executable provides two commands, `serve` and `build`.

- <kbd>iles serve</kbd>: Starts the development server
- <kbd>iles build</kbd>: Creates a production build of the site

It's highly recommended to add the following shortcuts to your `package.json`:

```json
  "scripts": {
    "dev": "iles --open",
    "build": "iles build",
    "preview": "vite preview --open"
  },
```

- <kbd>npm run dev</kbd>: Starts the development server
- <kbd>npm run build</kbd>: Creates a production build of the site
